# Week 3 실습 기록

## 목표

플레이어 캐릭터를 키보드로 움직이기 구현

## AI 대화 기록

**Q1: 아래 pygame 코드에 방향키 입력을 추가해줘 상하좌우 방향키/WASD를 누르면 그에 맞는 방향으로 원이 7픽셀씩 움직이게 해줘**

- AI 답변: pygame.key.get_pressed()로 키보드 입력 처리

적용 결과: 방향키 + WASD로 원이 이동 가능

배운 점: 키보드나 방향기로 플레이어를 움직이는 방법을 배움.

**Q2: 현재 코드에 경계 처리를 추가해줘 원이 화면 밖으로 나가지 못하게 해줘**

- AI 답변: pygame.key.get_pressed()로 키보드 입력 처리

적용 결과: 화면 밖으로 못 나가게 제한

배운 점: 화면 밖으로 플레이어가 못 나가게 하는 법을 배움.
